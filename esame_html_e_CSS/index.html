SKILLS

COMPETENZE ACQUISITE

Html 5

HTML5, erede del linguaggio del web, introduce funzioni multimediali integrate, grafica avanzata e semantica migliorata, rendendolo uno strumento potente e versatile per la creazione di siti web moderni e coinvolgenti.

Css 3

CSS3, erede del linguaggio di styling per il web, introduce una miriade di potenti strumenti per il design responsivo, animazioni fluide, effetti visivi accattivanti e tipografie personalizzate.

JavaScript

JavaScript, per dirla in parole semplici, è come un coltellino svizzero per il web. Aggiunge interattività e dinamismo alle pagine statiche HTML, rendendole vive e coinvolgenti.

php

PHP, acronimo di Hypertext Preprocessor, è un linguaggio di programmazione lato server ampiamente utilizzato per lo sviluppo web. permette di creare siti web dinamici e interattivi, integrandosi perfettamente con HTML e CSS. La sua natura open-source lo rende accessibile a tutti, con una vasta community pronta a supportare.
